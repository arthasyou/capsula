# Capsula-PKI 开发任务清单

## 📋 项目概览
基于 Rust 的完整 PKI (公钥基础设施) 实现，支持完整的证书生命周期管理。

## 🎯 核心功能模块

### ✅ 已完成模块
- [x] **CSR 模块** (`src/ra/csr.rs`) - 证书签名请求
  - [x] CSR 生成和解析
  - [x] PEM/DER 格式支持
  - [x] 签名验证
- [x] **Cert 模块** (`src/ra/cert.rs`) - 证书管理
  - [x] 证书生成（从CSR）
  - [x] 自签名证书
  - [x] X.509 格式支持
  - [x] 证书验证
